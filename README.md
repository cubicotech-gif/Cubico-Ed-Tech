# Cubico Technologies — EdTech Website

Official website for Cubico Technologies, a Karachi-based EdTech agency.

## Pages

- `index.html` — Homepage
- `services.html` — Services
- `portfolio.html` — Portfolio
- `about.html` — About Us
- `contact.html` — Contact

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. No config needed — Vercel auto-detects HTML
4. Click Deploy ✓
